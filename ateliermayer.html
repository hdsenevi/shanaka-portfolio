Introduction

‘Atelier Mayer’ is an iOS augmented Reality application showcasing an interactive sneak preview of the life of Madame Mayer, inviting you to discover her luxury vintage pieces using augmented reality.

Technical Details

Languages C#, Objective-C.
Libraries Vuforia SDK, iOS SDK, Unity 3d runtime, .Net 2.0, Sprite Manager 2, EZGUI, Prime31 plugins.
Tools Unity 3d, XCode, Photoshop.
My contribution I was responsible for all the code that went into making the application including core app framework and graphical user interface.
